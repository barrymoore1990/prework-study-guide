Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • html lang="en" is for English
  • title displays above the page
  • h1 is first title that displays on the page & headers should be ordered in logical order of importance for SEO
  • img src is to display an image and the alt text gives a description of the image for accessibility & SEO

CSS

  • CSS = cascading style sheets
  • It's like the decoration of a house after HTML is the foundations
  • It's to help the website look good

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git should be used for all development files
  • Allows you to code offline and upload when ready
  • Allows you to make changes in a branch as to avoid changing a live site
  • Branches also allow you to submit work for someone else to sign off

Javascript

  • Used so users can interact with a website
  • Like the doors and windows opening in a house
